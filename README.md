# Formulário Acessível

## Passos

1.  Criar o  HTML semântico permitindo navegação com Tab e melhor escanabilidade por parte do navegador

2. Adicionar atributos ARIA, para dar contexto aos leitores de tela, como:
- `aria-labelledby`
- `aria-required`
- `aria-describedby`
- `aria-label`  

3. Aplicar CSS simples só para organizar informações.
